FAQ

Is my data stored on-chain?
No, all user data is encrypted and stored on IPFS/Arweave. Only access controls live on-chain.

Can I revoke access?
Yes, you can revoke or rotate keys at any time through the Vault Dashboard.

Do I need to pay to use Organic Networks?
Basic vaults will be free for individuals. Monetization is opt-in and ethical.
